### 🚀 About Me

I build backend systems and dig into the layers most people don't — syscalls, namespaces, container internals. Most recently that meant writing **[PyCrate](https://github.com/Somshubhro07)**, a container runtime from scratch in Python, talking to the Linux kernel directly through `ctypes`.

- 🐍 Python-first, but comfortable across the JS/TS ecosystem when the project calls for it
- 🐧 Into Linux internals — namespaces, cgroups, OverlayFS, the stuff Docker is built on
- 🧠 Spend a good chunk of time evaluating AI-generated code for correctness and reasoning quality
- 💬 Ask me about backend architecture, Linux internals, or container runtimes
- ⚡ Fun fact: I'm left-handed

### 🌟 Featured Projects

**🐳 PyCrate** — *Python · ctypes · cgroups v2 · OverlayFS · FastAPI*
A container runtime built from scratch, talking to the Linux kernel directly — namespaces, cgroups, OverlayFS, and a seccomp filter, no Docker underneath.

**📈 Predictive Sales Dashboard** — *Python · Pandas · Scikit-learn*
Forecasts sales trends from raw data for non-technical business owners, ~80% accuracy on test data.

**📊 Stock Market Analysis Tool** — *Python · BigQuery*
Pulls historical stock data and visualizes price trends, volatility, and entry signals.

**📦 Shop-Sync** — *Node.js · Express · MongoDB · PostgreSQL*
Inventory management with role-based access, tracking 500+ SKUs with indexed queries.

**🍵 Tea E-Commerce Platform** — *MERN · Tailwind CSS*
Full storefront — catalog, cart, checkout — sub-2s load times.

**🏠 Airbnb Clone** — *MERN Stack*
